# Poker+

Калькулятор покерных решений для live-игры в Техасский Холдем.

## Возможности

- **Расчёт эквити** — Монте-Карло симуляция (preflop/flop/turn/river)
- **Аутсы и улучшение** — подсчёт аутов на флеш/стрит, правило 4/2
- **Название руки** — Топ-пара, Сет, Флеш, Стрейт и т.д.
- **Шансы банка** — pot odds, SPR, рекомендация (fold/call/raise)
- **Агрессия оппонента** — учитывает тип оппонента и его действие
- **Бет-сайзинг** — кнопки ⅓, ½, ⅔, Pot
- **Сессия** — трекинг стека, статистика решений, история раздач

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Монте-Карло на клиенте (Web Worker-ready)

## Запуск

```bash
npm install
npm run dev
```

Открой `http://localhost:3000`.

## Использование

1. Настрой сессию (стек, бай-ин, количество игроков)
2. Введи свою руку, позицию, стадию и борд
3. Укажи размер банка и ставку оппонента
4. Получи рекомендацию с % эквити, pot odds, аутсами и объяснением
5. Запиши результат (фолд/колл/рейз) для статистики
