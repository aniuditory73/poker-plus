# Poker+

Калькулятор покерных решений для live-игры в Техасский Холдем.

## Возможности

- **Range-сетка 13×13** — визуализация всех 169 стартеров с цветовой маркировкой (зелёный/жёлтый/красный по equity)
- **Расчёт эквити** — Монте-Карло симуляция (preflop/flop/turn/river) с учётом карт оппонента
- **Аутсы и улучшение** — подсчёт аутов на флеш/стрит, правило 4/2, название текущей руки
- **Pot odds & SPR** — шансы банка, stack-to-pot ratio, рекомендация (fold/call/raise)
- **Агрессия оппонента** — учёт типа и действия оппонента
- **Бет-сайзинг** — кнопки ⅓, ½, ⅔, Pot с автозаполнением
- **Графики сессии** — стек/выигрыш-проигрыш по раздачам (SVG, тултипы при наведении)
- **Статистика** — количество раздач, fold/call/raise, P&L за сессию
- **История раздач** — полная история с возможностью очистки
- **Тёмная/светлая тема** — переключение с сохранением в localStorage

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Монте-Карло симуляция на клиенте (Web Worker-ready)

## Запуск

```bash
npm install
npm run dev
```

Открой `http://localhost:3000`.

## Использование

1. Настрой сессию: бай-ин, стек, количество игроков
2. Выбери руку через Range-сетку → выбери масти
3. Укажи позицию, стадию, борд, банк и ставку оппонента
4. Получи рекомендацию с % эквити, pot odds, аутсами и объяснением
5. Запиши реальный исход раздачи (+$ / −$) — появится график и статистика
